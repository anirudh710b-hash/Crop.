# 🌱 Crop Recommendation System

A Machine Learning project that recommends the most suitable crop based on environmental and soil conditions. This project helps farmers and agricultural researchers make smarter farming decisions using data-driven predictions.

---

## 📌 Project Overview

The Crop Recommendation System uses Machine Learning algorithms to predict the best crop to grow based on several factors such as:

* Nitrogen level
* Phosphorus level
* Potassium level
* Temperature
* Humidity
* pH value
* Rainfall

The system is designed to support smart agriculture and improve crop productivity.

---

## 🚀 Features

* 🌾 Predicts suitable crops using ML algorithms
* 📊 Data preprocessing and analysis
* 🧠 Trained Machine Learning model
* 🌐 User-friendly Flask web interface

* ⚡ Fast prediction system

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/anirudh710b-hash/Crop..git
cd Crop.
```

### 2️⃣ Create Virtual Environment (Optional)

```bash
python -m venv venv
```

### 3️⃣ Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Start the Flask server:

```bash
python app.py
```

Now open your browser and visit:

```bash
http://127.0.0.1:5000/
```
